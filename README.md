# Android-Hustoo
华科校园软件Hustoo Hust
##简介/Abstract

一款华科校园软件，含教务通知，华科社区，天气预报(Hust)，每日美文鸡汤音乐分享等。 
<br/><br/>
A Hust campus software, including educational administration notice, Hust community, 
<br/><br/>
weather forecast, daily notes chicken stock share music, etc.
<br/><br/>
平台/Platform：AS
